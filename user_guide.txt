YDL Studio 사용법
===================

본 프로그램은 **yt‑dlp** 라이브러리를 이용하여 YouTube를 비롯한 여러 사이트에서 동영상 및 음원, 자막 등을 손쉽게 내려받을 수 있도록 도와주는 그래픽 인터페이스입니다. 초보자도 쉽게 사용할 수 있도록 설계되었습니다.

사전 준비
---------

1. 이 앱은 내부적으로 **yt‑dlp**와 **FFmpeg**에 의존합니다. 프로그램 실행 전에 Python 환경에 `yt-dlp`를 설치해 두어야 합니다. Windows에서 실행 파일(.exe) 버전을 사용하면 별도의 Python이 필요 없습니다.
2. 고화질 영상을 병합하거나 오디오 추출, 썸네일/메타데이터 삽입 기능을 이용하려면 동일한 폴더에 `ffmpeg.exe`와 `ffprobe.exe`를 추가해 주세요. (※ FFmpeg는 https://ffmpeg.org 에서 받을 수 있습니다.)
3. 권한이 필요한 영상(로그인 또는 성인 인증 영상)인 경우, 웹 브라우저의 쿠키를 함께 불러와야 합니다. 현재 버전은 기본 탭에서 쿠키 불러오기를 지원하지 않으므로 `--cookies-from-browser` 옵션을 수동으로 활용하거나 다운로드 가능한 yt‑dlp 방법을 참고하시기 바랍니다.

사용 방법
---------

1. 프로그램을 실행하면 첫 화면에 **Video URLs** 입력란이 보입니다. 한 줄에 하나씩 원하는 동영상 주소를 붙여넣습니다. 여러 개를 동시에 입력할 수 있습니다.
2. "**Save to**" 옆의 **Browse…** 버튼을 눌러 내려받을 폴더를 선택합니다. 기본값은 현재 실행 위치입니다.
3. "**Preset**" 드롭다운 메뉴에서 원하는 품질 또는 작업 유형을 선택합니다.
   - **Best Video**: 자동으로 가장 높은 해상도와 음질의 조합을 선택합니다.
   - **1080p MP4**: 1080p 이하의 MP4 형식을 선택합니다.
   - **Audio Only (MP3)**: 음원만 추출하여 MP3 형식으로 변환합니다.
   - **Subtitles Only**: 영상을 다운로드하지 않고 자막만 저장합니다.
4. 자막을 저장하고 싶다면 "**Sub languages**"에 `ko,en` 처럼 언어 코드를 쉼표로 구분하여 입력합니다. 자동 자막을 포함하려면 아래 **Auto subtitles** 체크박스를 선택하세요.
5. 옵션에서 필요에 따라 다음을 선택합니다.
   - **Embed thumbnail**: 다운로드한 파일에 썸네일 이미지를 삽입합니다.
   - **Embed metadata**: 제목, 설명 등 메타데이터를 삽입합니다.
   - **Auto subtitles**: 자동 생성된 자막을 함께 가져옵니다.
6. 모든 설정이 끝나면 **Start** 버튼을 눌러 다운로드를 시작합니다. 진행 상황은 하단 **Log** 창에 표시됩니다.

* Tip: URL 입력 창을 마우스로 오른쪽 클릭하면 "Paste" 메뉴가 나타나서 복사한 주소를 붙여넣을 수 있습니다. Ctrl+V 키를 사용할 수도 있습니다.
7. 진행률 표시줄은 현재 다운로드하는 파일의 진행 상황을 보여 줍니다. 여러 URL을 입력한 경우 순차적으로 진행됩니다.
8. **How to Use** 버튼을 클릭하면 이 문서의 전체 내용이 새 창에 표시됩니다.

주의 사항
---------

* 일부 사이트는 이용약관에 따라 다운로드가 금지되어 있을 수 있습니다. 콘텐츠를 내려받기 전에 해당 사이트의 정책과 현지 법규를 준수하세요.
* 로그인 또는 연령 인증이 필요한 콘텐츠는 브라우저 쿠키를 가져와야만 다운로드할 수 있습니다. 자세한 사용법은 yt‑dlp 공식 문서를 참고하세요.
